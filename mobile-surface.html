Mobile surface — design logic
Interrupt model
Agent pushes, human gates
Agents find and prioritize. Push arrives on lock screen with one-sentence context. Human approves or denies without opening the full workstation.
Three decisions
OK · Deny · Defer
OK executes the agent's top proposal. Deny returns the item to the queue for human rework. Defer pushes it 24h and re-scores priority.
Proposal detail
Tap through for context
Each action links to ranked resolution options with confidence scores. All data from Cognite + IMF attribute validation — no re-entry needed.
Desktop link
Deep work stays on desktop
Mobile never shows raw data tables, P&IDs or graph views. Those open on the desktop workstation. Mobile is purely the interrupt and approval layer.
